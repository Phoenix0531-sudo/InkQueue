## 功能

- 🖥️ `server/` Node 服务
- 📱 `android/` 客户端实验
- 🧪 server 路径 npm test CI
- 🔧 scripts/ + tasks 运维辅助

## 快速开始

### 安装

```bash
git clone https://github.com/Phoenix0531-sudo/InkQueue.git
cd InkQueue/server
npm install
npm test
```

### 使用

启动命令见 `server/` README / scripts。密钥勿入库。
