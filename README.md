- 🧪 npm test CI for server path

## Get started

### Install

```bash
git clone https://github.com/Phoenix0531-sudo/InkQueue.git
cd InkQueue/server
npm install
npm test
```

### Usage

See `server/` README / scripts for start commands. Keep tokens out of git.
